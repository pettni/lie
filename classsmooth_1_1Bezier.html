Detailed Description

template<std::size_t N, LieGroup G>
class smooth::Bezier< N, G >

Bezier curve on [0, 1].

Template Parameters
NPolonimial degree of curve.
GLieGroup type.

The curve is defined by

\[ g(t) = g_0 * \exp(\tilde B_1(t) v_1) * ... \exp(\tilde B_N(t) v_N) \]

where \(\tilde B_i(t)\) are cumulative Bernstein basis functions and \(v_i = g_i \ominus g_{i-1}\) are the control point differences.
