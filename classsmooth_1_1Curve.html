Detailed Description

template<LieGroup G>
class smooth::Curve< G >

Single-parameter Lie group-valued function.

A Curve is a continuous function \( x : \mathbb{R} \rightarrow \mathbb{G} \) defined on an interval \( [0, T] \) such that \( x(0) = e \).

Internally a Curve is represented via third-order polynomials, similar to a PiecewiseBezier of order 3.
