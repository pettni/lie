Detailed Description

template<std::size_t N, LieGroup G>
class smooth::PiecewiseBezier< N, G >

Piecewise curve built from Bezier segments.

The curve is given by

\[ \mathbf{x}(t) = p_i \left( \frac{t - t_i}{t_{i+1} - t_{i}} \right) \]

for \( t \in [t_i, t_{i+1}]\) where \(p_i\) is a Bezier curve on \([0, 1]\).
