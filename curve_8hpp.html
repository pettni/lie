Reparameterize a curve to approximately satisfy velocity and acceleration constraints.

If \( x(\cdot) \) is a Curve, then this function generates a function \( s(t) \) the reparamterized curve \( x(s(t)) \) has body velocity bounded between vel_min and vel_max, and body acceleration bounded between acc_min and acc_max.

Parameters
curveCurve \( x(t) \) to reparameterize.
vel_min,vel_maxvelocity bounds, must be s.t. vel_min < 0 < vel_max (component-wise).
acc_min,acc_maxacceleration bounds, must be s.t. acc_min < 0 < acc_max (component-wise).
start_veltarget value for \( s'(0) \) (must be non-negative).
end_veltarget value for \( s'(t_{max}) \) (must be non-negative).
slower_onlyresult is s.t. \( s'(t) <= 1 \).
dttime discretization step (smaller step gives a more accurate solution).
epsparameter that controls max and min velocities, and step size for unconstrained segments
Note
It may not be feasible to satisfy the target boundary velocities. In those cases the resulting velocities will be lower than the desired values.
